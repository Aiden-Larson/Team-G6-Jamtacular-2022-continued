
# Entity RGBA codes


| Alpha        | Meaning                |
| ------------ | ---------------------- |
| 0            | No block/entity        |
| 1            | Spawn Block            |
| 2            | Props                  |
| 3            | Pickups                |
| 4            | Enemies                |


### Props
| Entity       | Code                   |
| ------------ | ---------------------- |
| pillar       | 0, 0, 0, 2             |
| table        | 0, 0, 1, 2             |
| etc          | 0, 0, 2, 2             |
